# Game Project

An open source game built with [Godot Engine 4.6](https://godotengine.org/).

## Getting Started

### Prerequisites
- [Godot Engine 4.6](https://godotengine.org/download/) or later

### Running the Project
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/game-project.git
   ```
2. Open Godot Engine
3. Click **Import** and select the `project.godot` file
4. Press **F5** or click the Play button to run
